設置方法

ページ内埋め込み

表示例のようなGoogleマップ版ルート地図を表示するには、対象サイトに下記のディレクトリ構成図のようにファイルを設置し、対象ページに表示のためのコードを挿入します。

ディレクトリ構成図
.
├── your_page.html（対象ページ）
├── gmap_ex2.html
└── share
    └── routemap.geojson
対象ページに挿入するコード
<iframe width="640" height="480" style="border:none;"
 src="gmap_ex2.html?lat=35.913163&lon=138.721400&zoom=15&url=share/routemap.geojson">
</iframe>

その他の表示方式（別窓ポップアップ）については、設置方法（共通）を参照して下さい。

ソースコード

本サイトで説明している各種ルート地図やGPX2GeoJSONのソースコードはGitHubで管理しているので、ソースコードの入手にはgit（バージョン管理システム）があると便利です。gitによる入手方法については、設置方法（共通）を参照して下さい。gmap_ex2.html は下記のリンクからもダウンロードできます。'[YOUR API KEY]'の個所に各自がGoogle Cloud Platformから取得したAPI Keyを記入して、設置して下さい。

gmap_ex2.html (ダウンロード)






変更個所（36行目）


<script src="https://maps.googleapis.com/maps/api/js?key=[YOUR API KEY]&language=ja&region=JP"></script>
